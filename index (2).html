Tentang TKJ

Teknik Komputer dan Jaringan (TKJ) adalah bidang keahlian yang mempelajari tentang cara merakit komputer, menginstal program sistem operasi, serta membangun dan mengonfigurasi jaringan komputer lokal (LAN) maupun luas (WAN). Saya memiliki passion mendalam dalam memahami bagaimana lalu lintas data beroperasi dan mengamankan infrastruktur jaringan.

Server Administration

Pengelolaan OS Server (Linux Debian/Ubuntu), Web Server, DHCP, dan DNS Server.

Hardware & Troubleshooting

Perakitan PC, perawatan perangkat keras, dan diagnosa kerusakan komputer.

Network Security

Dasar-dasar keamanan jaringan, Firewall, monitoring bandwidth, dan proteksi jaringan.
