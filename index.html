Tentang Jurusan TKJ

Teknik Komputer dan Jaringan (TKJ) adalah bidang keahlian yang mempelajari cara merakit komputer, menginstalasi jaringan lokal (LAN) maupun luas (WAN), mengonfigurasi perangkat komunikasi seperti Router & Switch, serta mengelola sistem keamanan dan server komputer.

Keahlian & Kompetensi

Jaringan Komputer

Mampu membuat perkabelan UTP/STP, konfigurasi Mikrotik, IP Addressing, Subnetting, dan Routing.

Administration Server

Konfigurasi Server Linux/Windows, Web Server, DHCP, DNS, serta File Server.

Hardware & Perakitan

Merakit komponen PC, troubleshooting masalah teknis hardware, serta instalasi OS.

Keamanan Jaringan

Memahami konsep dasar Firewall, VPN, dan manajemen proteksi data dari akses tak berizin.
